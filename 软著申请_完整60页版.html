1. 软件基本情况

1.1 软件名称

我的高考志愿模拟器软件 V1.0

1.2 软件版本

V1.0

1.3 运行环境

  • 操作系统：Android、iOS、Windows、macOS
  • 运行平台：微信小游戏、抖音小游戏、TapTap
  • 引擎版本：团结引擎（Tuanjie Engine）2D URP
  • 屏幕比例：竖版 9:16（750×1334）

1.4 主要功能概述

"我的高考志愿模拟器软件 V1.0"是一款基于团结引擎开发的人生模拟养成类小游戏。玩家从高三入学开始，经历高中三年学习生活、天赋选择、高考出分、志愿填报、大学生活、职场发展，最终抵达人生结局。游戏融合了天赋系统、性别区分、人格演化、多结局、多周目传承等多种玩法机制。

1.5 核心玩法亮点

  • 性别选择系统：首次登录选择性别，不同性别显示不同事件插图
  • 天赋系统：12种天赋三选一，贯穿游戏全流程
  • 人格系统：8种高考人格，根据选择动态演化
  • 多周目传承：4级既视感系统，多代属性继承
  • 志愿填报：全国44所高校数据，三层梯度填报策略

2. 软件功能详细说明

2.1 性别选择系统

玩家首次进入游戏时，需选择角色性别（男性/女性）。性别选择后，游戏内所有事件插图将根据性别加载对应图片：男性角色显示男性事件插图，女性角色显示女性事件插图。性别信息存储于玩家存档中（PlayerState.gender），伴随整个游戏流程。存档时性别信息一同保存，确保每次读取存档后性别状态正确。

2.2 天赋系统

游戏内置12种天赋，玩家在高三入学时通过三选一方式获得天赋。天赋值贯穿游戏全流程，影响各阶段属性计算。

2.3 高中三年阶段

玩家进入高三后，通过选择事件选项来塑造角色属性。游戏包含20+个高中事件，覆盖学习类（月考、期末复习、模拟考试）、情感类（收到情书、与同学吵架）、校园类（运动会、班级篮球赛、志愿者活动）、家庭类（家长会、家里出状况）、关键抉择类（文理分科、考前焦虑）等类型。每个事件提供多个选项，各选项对智力(Intellect)、心态(Mental)、社交(Social)、健康(Health)四项属性产生不同影响。游戏通过时间线展示高三三年六个学期的进程。

2.4 高考出分阶段

高三结束进入高考阶段。系统根据玩家三年积累的属性值（智力、心态）、天赋加成和省份因素，计算最终高考成绩。成绩计算引擎(TalentEngine)综合以下因素：智力基础分（0-100映射到0-300）、心态波动分（心态影响考试稳定性）、天赋加成（不同天赋有不同加成系数）、省份难度系数（各省录取分数线不同）。最终成绩显示采用动画效果，从0分滚动到最终分数。

2.5 志愿填报阶段

高考出分后进入志愿填报阶段。系统提供全国44所高校数据，涵盖985工程高校（清华、北大、复旦、上海交大等）、211工程高校（北京邮电、中央财经等）、双一流高校、普通本科院校。每所高校包含：id、name、tags(985/211标签)、province(所属省份)、type(综合/理工/师范/财经等)、rank(排名)、features(特色)、hot_majors(热门专业)、scores(各省各年录取分数线)。志愿填报采用三层梯度策略：冲一冲（高于考生成绩5-15分的院校）、稳一稳（等于或略低于考生成绩的院校）、保一保（低于考生成绩10分以上的院校）。录取结果根据考生成绩与各院校分数线匹配计算。

2.6 大学四年阶段

进入大学后，根据考生录取的专业方向，加载对应的事件集：计算机方向（college_cs.json，5个事件）、法学方向（college_law.json，4个事件）、医学方向（college_med.json，5个事件）、商科方向（college_biz.json，4个事件）。大学四年期间，玩家需要做出毕业方向选择：考研深造、直接就业、出国留学、创业。不同选择将影响人生阶段的职业路径和收入水平。

2.7 人生阶段

毕业方向选择后进入人生阶段。根据毕业选择，人生事件集将有所不同。玩家将在22-30岁期间经历多个关键人生事件，每个事件提供3个选项，影响月收入变化、职业发展、家庭状态、满意度评分。

2.8 结局与多周目传承

游戏设置多种结局类型，根据月收入和满意度综合判定：完美结局（月入≥30k且满意度≥4）、良好结局（月入≥15k且满意度≥3）、苦涩结局（满意度≤2）、普通结局（其他情况）。多周目传承机制允许玩家将当前人生的属性加成（按10-15%比例）遗传给下一代，并解锁既视感(Déjà Vu)能力：既视感Lv1（关键选项有微弱提示）、既视感Lv2（部分事件结果预览）、既视感Lv3（完美记忆所有选择）、既视感Lv4（解锁隐藏剧情线）。

第二部分：技术设计说明书

一、系统架构概述

1.1 整体架构

本软件采用Unity/Tuanjie Engine 2D URP架构，使用C#进行开发，遵循分层架构设计原则。

1.2 核心设计模式

  • 单例模式(Singleton)：所有Manager和Engine类使用Instance单例模式，确保全局唯一实例，便于跨场景访问
  • 观察者模式(Observer)：GameManager通过event事件机制实现状态变更通知，UI层订阅事件更新界面
  • 工厂模式：DataLoader作为数据工厂统一创建各类游戏数据对象，提供统一的加载接口
  • 策略模式：事件系统支持多种交互策略（choice/merge/sort/slider/timing/dialog），通过interactionType字段动态选择处理方式

二、核心模块设计

2.1 游戏管理器（GameManager）

游戏总控制器，负责游戏生命周期管理。主要职责包括：管理游戏全局状态（isGameRunning, isPaused）、协调各Engine和Manager之间的通信、处理场景切换逻辑、管理玩家数据读写。核心方法包括：StartNewGame()（初始化新游戏流程）、ContinueGame()（从存档恢复游戏）、ChangePhase()（阶段切换）、SaveGame()（保存游戏数据）、CalculateGaokaoScore()（调用ScoreEngine计算高考成绩）、StartNewGamePlus()（二周目新游戏创建）。

2.2 游戏状态管理器（GameStateManager）

独立于GameManager的状态管理组件，生命周期贯穿所有场景。主要职责包括：维护GameSaveData数据结构、提供QuickSave/QuickLoad快速存取接口、管理玩家性别（Gender）信息、协调存档与引擎间的数据同步。与GameManager的区别在于：GameManager专注游戏流程控制，GameStateManager专注数据持久化。

2.3 事件引擎（EventEngine）

驱动游戏进程的核心引擎，负责事件的获取、选择和结算。核心流程：获取事件 → 检查触发条件 → 检查既视感 → 展示事件 → 玩家交互选择 → 调用ResolveEvent计算结果 → 更新属性 → 记录事件完成 → 获取下一事件 → 循环。核心方法包括：SetPlayerState()（注入玩家状态）、GetNextEvent()（获取下一事件）、ResolveEvent()（结算事件结果）、LoadEventsFromStreamingAssets()（从StreamingAssets加载事件）。

2.4 天赋引擎（TalentEngine）

管理12种天赋的抽卡、选择和效果计算。核心方法包括：DrawTalents()（抽卡获取候选天赋）、SelectTalent()（玩家选择天赋）、GetCurrentTalent()（获取当前天赋）、ApplyTalentEffect()（应用天赋效果）、ApplyGaokaoModifiers()（应用高考分数修正）、ApplyHighschoolModifiers()（应用高中属性修正）。

2.5 成绩计算引擎（ScoreEngine）

根据玩家属性和天赋计算高考成绩。计算公式：最终成绩 = 智力基础分 × 2.5 + 心态加成 + 健康加成 + 随机波动 + 天赋修正。核心方法包括：CalculateScore()（主计算入口）、SplitSubjects()（拆分单科成绩）、CalculateProbability()（计算录取概率）。

2.6 人格引擎（PersonalityEngine）

根据玩家属性计算高考人格类型。人格判定规则：取Intellect/Mental/Social/Health四个维度的加权分数，选择最高分对应的人格类型。人格类型决定游戏内部分对话和结局倾向。核心方法包括：CalculatePersonality()（计算人格）、GetPersonalityScores()（获取人格评分）。

2.7 既视感引擎（DejaVuEngine）

二周目记忆系统，提供前世记忆和选择提示。既视感等级（L1-L4）：L1微弱直觉（仅在关键选项时显示模糊提示）、L2记忆片段（显示选项可能导致的属性变化方向）、L3清晰记忆（显示每个选项的历史结果）、L4完美直觉（解锁隐藏剧情选项）。核心方法包括：CheckDejaVu()（检查是否有前世记忆）、RecordMemory()（记录当前游戏记忆）、LoadMemories()（加载前世记忆）。

2.8 传承引擎（InheritEngine）

多代传承系统，将当前人生的属性遗传给下一代。传承内容包括：属性加成（Intellect/Mental/Social/Health各传承10-15%）、记忆天赋（既视感等级）、校友院校（解锁特定招生优惠）、体验过的专业（解锁专业相关内容）。核心方法包括：StartNewGeneration()（开始新世代）、GetMaxDejaVuLevel()（获取最大既视感等级）、IsAlumniCollege()（检查是否校友院校）。

using System;
using System.Collections.Generic;
using UnityEngine;
using UnityEngine.SceneManagement;

public class GameManager : MonoBehaviour
{
    public static GameManager Instance { get; private set; }

    private PlayerState currentPlayer;
    private GamePhase currentPhase;
    private bool isGameRunning;
    private bool isPaused;

    private List gameSceneNames = new List
    {
        "Home", "Province", "HighSchool", "Event",
        "Gaokao", "Zhiyuan", "College", "Life", "Result"
    };

    public event Action OnPlayerStateChanged;
    public event Action OnPhaseChanged;
    public event Action OnGameStarted;
    public event Action OnGameEnded;

    public PlayerState CurrentPlayer => currentPlayer;
    public GamePhase CurrentPhase => currentPhase;
    public bool IsGameRunning => isGameRunning;
    public bool IsPaused => isPaused;

    void Awake()
    {
        if (Instance == null)
        {
            Instance = this;
            DontDestroyOnLoad(gameObject);
        }
        else
        {
            Destroy(gameObject);
            return;
        }
        Initialize();
    }

    private void Initialize()
    {
        currentPhase = GamePhase.Home;
        isGameRunning = false;
        isPaused = false;
        Debug.Log("[GameManager] 游戏管理器初始化完成");
    }

    public void StartNewGame()
    {
        Gender savedGender = Gender.Male;
        if (SaveManager.Instance != null)
        {
            var existing = SaveManager.Instance.GetCurrentPlayer();
            if (existing != null) savedGender = existing.gender;
        }

        if (SaveManager.Instance != null)
        {
            SaveManager.Instance.CreateNewGame();
            currentPlayer = SaveManager.Instance.GetCurrentPlayer();
            if (currentPlayer != null) currentPlayer.gender = savedGender;
        }
        else
        {
            currentPlayer = new PlayerState();
        }

        isGameRunning = true;
        currentPhase = GamePhase.Home;
        LoadGameData();

        OnPlayerStateChanged?.Invoke(currentPlayer);
        OnPhaseChanged?.Invoke(currentPhase);
        OnGameStarted?.Invoke();

        Debug.Log("[GameManager] 开始新游戏");
        LoadScene("Home");
    }

    public void ContinueGame()
    {
        if (SaveManager.Instance != null && SaveManager.Instance.LoadGame())
        {
            currentPlayer = SaveManager.Instance.GetCurrentPlayer();
            isGameRunning = true;
            LoadGameData();

            OnPlayerStateChanged?.Invoke(currentPlayer);
            OnPhaseChanged?.Invoke(currentPlayer.currentPhase);

            Debug.Log("[GameManager] 继续游戏");
            LoadScene(currentPlayer.currentPhase.ToString());
        }
    }

    public void ChangePhase(GamePhase newPhase)
    {
        if (currentPhase == newPhase) return;

        currentPhase = newPhase;
        currentPlayer.currentPhase = newPhase;

        OnPhaseChanged?.Invoke(newPhase);
        Debug.Log($"[GameManager] 阶段切换: {newPhase}");

        LoadScene(newPhase.ToString());
    }

    public void SaveGame(int slotIndex = 0)
    {
        if (currentPlayer == null)
        {
            Debug.LogWarning("[GameManager] 没有玩家数据可保存");
            return;
        }

        currentPlayer.lastPlayTime = DateTime.Now;

        if (SaveManager.Instance != null)
        {
            SaveManager.Instance.SetCurrentPlayer(currentPlayer);
            SaveManager.Instance.SaveGame(slotIndex);
        }
    }

    public int CalculateGaokaoScore()
    {
        if (ScoreEngine.Instance != null)
        {
            string talentId = TalentEngine.Instance?.GetCurrentTalent()?.id ?? "";
            return ScoreEngine.Instance.CalculateScore(currentPlayer, talentId);
        }
        return 400;
    }

    public void StartNewGamePlus()
    {
        Gender savedGender = Gender.Male;
        var existing = SaveManager.Instance?.GetCurrentPlayer();
        if (existing != null) savedGender = existing.gender;

        SaveManager.Instance?.CreateNewGame();
        currentPlayer = SaveManager.Instance?.GetCurrentPlayer() ?? new PlayerState();
        currentPlayer.gender = savedGender;
        currentPlayer.hasPastLifeMemory = true;

        isGameRunning = true;
        ChangePhase(GamePhase.HighSchool);

        Debug.Log("[GameManager] 开始二周目");
    }
}

using UnityEngine;
using System.Collections;
using System.Collections.Generic;
using System.Linq;

public class EventEngine : MonoBehaviour
{
    private List eventPool = new List();
    private PlayerState playerState;

    public void LoadEventPool(List events)
    {
        eventPool = events ?? new List();
        Debug.Log($"[EventEngine] 加载 {eventPool.Count} 个事件");
    }

    public void SetPlayerState(PlayerState state)
    {
        playerState = state;
    }

    public GameEvent GetNextEvent()
    {
        if (playerState == null)
        {
            Debug.LogWarning("[EventEngine] playerState 为空");
            return null;
        }

        if (eventPool.Count == 0)
        {
            Debug.LogWarning("[EventEngine] eventPool 为空");
            return null;
        }

        var candidates = eventPool
            .Where(e => !playerState.completedEvents.Contains(e.id))
            .Where(e => CheckCondition(e.triggerCondition))
            .ToList();

        if (candidates.Count == 0)
        {
            Debug.Log("[EventEngine] 没有可触发的事件");
            return null;
        }

        var weights = candidates.Select(e => CalculateWeight(e)).ToList();
        int totalWeight = weights.Sum();

        if (totalWeight <= 0)
        {
            return candidates[Random.Range(0, candidates.Count)];
        }

        int random = Random.Range(0, totalWeight);
        int cumulative = 0;
        for (int i = 0; i < candidates.Count; i++)
        {
            cumulative += weights[i];
            if (random < cumulative)
                return candidates[i];
        }

        return candidates[0];
    }

    private int CalculateWeight(GameEvent evt)
    {
        int weight = 10;
        if (evt.tags != null)
        {
            if (evt.tags.Contains("考试") && playerState.intellect > 70) weight += 5;
            if (evt.tags.Contains("情感") && playerState.social > 60) weight += 3;
            if (evt.tags.Contains("志愿") && playerState.grade >= 5) weight += 8;
        }
        return weight;
    }

    public EventOutcome ResolveEvent(string eventId, string choiceId)
    {
        var evt = eventPool.Find(e => e.id == eventId);
        if (evt == null)
        {
            Debug.LogWarning($"[EventEngine] 找不到事件: {eventId}");
            return null;
        }

        var outcomes = evt.GetOutcomeDict();
        if (!outcomes.ContainsKey(choiceId))
        {
            Debug.LogWarning($"[EventEngine] 事件 {eventId} 没有选择 {choiceId}");
            return null;
        }

        var outcome = outcomes[choiceId];
        playerState.ApplyChanges(outcome.GetStatChanges());
        playerState.completedEvents.Add(eventId);
        playerState.RecordChoice(eventId, choiceId);

        Debug.Log($"[EventEngine] 事件完成 {evt.name} -> {choiceId}: {outcome.narrative}");

        return outcome;
    }

    private bool CheckCondition(TriggerCondition cond)
    {
        if (cond == null) return true;
        if (cond.minGrade > 0 && playerState.grade < cond.minGrade) return false;
        if (cond.minIntellect > 0 && playerState.intellect < cond.minIntellect) return false;
        if (!string.IsNullOrEmpty(cond.requiredEvent) &&
            !playerState.completedEvents.Contains(cond.requiredEvent)) return false;
        return true;
    }
}

using UnityEngine;
using System.Collections.Generic;
using System.Linq;

public class TalentEngine : MonoBehaviour
{
    public static TalentEngine Instance { get; private set; }

    private List allTalents;
    private Talent currentTalent;

    void Awake()
    {
        if (Instance == null) { Instance = this; DontDestroyOnLoad(gameObject); }
        else { Destroy(gameObject); }
    }

    public void LoadTalents(List talents)
    {
        allTalents = talents ?? new List();
        Debug.Log($"[TalentEngine] 加载 {allTalents.Count} 个天赋");
    }

    public void SetCurrentTalent(Talent talent)
    {
        currentTalent = talent;
        Debug.Log($"[TalentEngine] 选中天赋: {talent?.name}");
    }

    public Talent GetCurrentTalent()
    {
        return currentTalent;
    }

    public List DrawTalents()
    {
        if (allTalents == null || allTalents.Count == 0)
        {
            Debug.LogWarning("[TalentEngine] 天赋池为空");
            return new List();
        }

        var pool = allTalents.Where(t => t.rarity != "legendary").ToList();
        var result = new List();

        for (int i = 0; i < 3 && pool.Count > 0; i++)
        {
            var weighted = pool.Select(t => new { Talent = t, Weight = GetRarityWeight(t.rarity) }).ToList();
            int totalWeight = weighted.Sum(w => w.Weight);
            int rand = Random.Range(0, totalWeight);
            int cumulative = 0;
            foreach (var w in weighted)
            {
                cumulative += w.Weight;
                if (rand < cumulative)
                {
                    result.Add(w.Talent);
                    pool.Remove(w.Talent);
                    break;
                }
            }
        }

        Debug.Log($"[TalentEngine] 抽卡结果: {string.Join(", ", result.Select(t => t.name))}");
        return result;
    }

    private int GetRarityWeight(string rarity)
    {
        return rarity switch
        {
            "common" => 60,
            "rare" => 30,
            "epic" => 10,
            _ => 10
        };
    }

    public int ApplyGaokaoModifiers(string talentId, int baseScore)
    {
        var talent = allTalents.Find(t => t.id == talentId);
        if (talent?.statModifiers?.gaokao == null) return baseScore;

        var mod = talent.statModifiers.gaokao;

        int score = baseScore + mod.baseBonus;
        score += Random.Range(-mod.volatility, mod.volatility + 1);

        if (Random.value < mod.critChance)
        {
            score += mod.critBonus;
            Debug.Log($"[TalentEngine] {talent.name} 爆分! +{mod.critBonus}");
        }

        return Mathf.Clamp(score, 200, 750);
    }

    public void ApplyHighschoolModifiers(string talentId, PlayerState state)
    {
        var talent = allTalents.Find(t => t.id == talentId);
        if (talent?.statModifiers?.highschool == null) return;

        var bonus = talent.statModifiers.highschool;
        var changes = new Dictionary();
        if (bonus.intellect != 0) changes["intellect"] = bonus.intellect;
        if (bonus.mental != 0) changes["mental"] = bonus.mental;
        if (bonus.social != 0) changes["social"] = bonus.social;
        if (bonus.health != 0) changes["health"] = bonus.health;
        state.ApplyChanges(changes);
    }
}

using UnityEngine;

public class ScoreEngine : MonoBehaviour
{
    public static ScoreEngine Instance { get; private set; }

    void Awake()
    {
        if (Instance == null) { Instance = this; DontDestroyOnLoad(gameObject); }
        else { Destroy(gameObject); }
    }

    public int CalculateScore(PlayerState state, string talentId)
    {
        float baseScore = 400 + state.intellect * 2.5f + Random.Range(-30f, 31f);
        float mentalBonus = state.mental * 0.3f;
        float healthBonus = state.health * 0.2f;
        float total = baseScore + mentalBonus + healthBonus;

        int rawScore = Mathf.RoundToInt(total);

        var talentEngine = TalentEngine.Instance;
        if (talentEngine != null)
            rawScore = talentEngine.ApplyGaokaoModifiers(talentId, rawScore);

        return Mathf.Clamp(rawScore, 200, 750);
    }

    public SubjectScores SplitSubjects(int totalScore)
    {
        float chinese = Random.Range(90, 121);
        float mathRatio = Random.Range(0.5f, 1f);
        float englishRatio = Random.Range(0.5f, 1f);
        float comprehensive = totalScore - chinese - mathRatio * 150 - englishRatio * 150;

        return new SubjectScores
        {
            chinese = Mathf.RoundToInt(chinese),
            math = Mathf.RoundToInt(mathRatio * 150),
            english = Mathf.RoundToInt(englishRatio * 150),
            comprehensive = Mathf.RoundToInt(comprehensive)
        };
    }

    public int CalculateProbability(int score, int minScore, int avgScore)
    {
        if (avgScore <= minScore) return 50;

        float prob = (float)(score - minScore) / (avgScore - minScore) * 100f;
        return Mathf.Clamp(Mathf.RoundToInt(prob), 5, 95);
    }

    public string GetProbabilityLabel(int probability)
    {
        if (probability >= 80) return "稳";
        if (probability >= 60) return "保";
        if (probability >= 30) return "冲";
        return "垫";
    }
}

public class SubjectScores
{
    public int chinese;
    public int math;
    public int english;
    public int comprehensive;

    }

    public DejaVuResult CheckDejaVu(string eventId, int maxLevel)
    {
        PlaythroughMemory bestMatch = null;
        for (int i = memories.Count - 1; i >= 0; i--)
        {
            var rec = memories[i].choiceRecords?.Find(r => r.eventId == eventId);
            if (rec != null) { bestMatch = memories[i]; break; }
        }

        if (bestMatch == null)
        {
            Debug.Log($"[DejaVuEngine] 事件 {eventId} 无前世记忆");
            return null;
        }

        var choice = bestMatch.choiceRecords.Find(r => r.eventId == eventId);
        int level = Mathf.Min(maxLevel, 4);

        var result = new DejaVuResult
        {
            eventId = eventId,
            previousChoice = choice.choiceId,
            hint = GenerateHint(choice, level),

public static class JsonHelper
{
    public static T[] FromJson(string json)
    {
        Wrapper wrapper = JsonUtility.FromJson>(json);
        return wrapper.Items;
    }

    public static string ToJson(T[] array)
    {
        Wrapper wrapper = new Wrapper();
        wrapper.Items = array;
        return JsonUtility.ToJson(wrapper);
    }

    public static string ToJson(T[] array, bool prettyPrint)
    {
        Wrapper wrapper = new Wrapper();
        wrapper.Items = array;
        return JsonUtility.ToJson(wrapper, prettyPrint);
    }

    [Serializable]
    private class Wrapper
    {
        public T[] Items;
    }

    public static string AddArrayWrapper(string json)
    {
        if (string.IsNullOrEmpty(json)) return "[]";
        if (json.TrimStart().StartsWith("[")) return json;
        return $"[{json}]";
    }

    public static string RemoveArrayWrapper(string json)
    {
        if (string.IsNullOrEmpty(json)) return json;
        string trimmed = json.Trim();
        if (trimmed.StartsWith("[") && trimmed.EndsWith("]"))
            return trimmed.Substring(1, trimmed.Length - 2);
        return json;
    }

    public Button confirmButton;
    public Text titleText;

    private List currentTalents;
    private Talent selectedTalent;

    public void ShowTalents(List talents)
    {
        currentTalents = talents;
        selectedTalent = null;
        confirmButton.interactable = false;
        
        foreach (Transform child in talentContainer) Destroy(child.gameObject);
        
        foreach (var talent in talents)
        {
            GameObject obj = Instantiate(talentCardPrefab, talentContainer);
            TalentCard card = obj.GetComponent();
            card.SetTalent(talent);
            card.OnSelect += OnTalentSelect;
        }
    }

    private void OnTalentSelect(Talent talent)
    {
        selectedTalent = talent;
        confirmButton.interactable = true;
        
        foreach (Transform child in talentContainer)
        {
            TalentCard card = child.GetComponent();
            card.SetSelected(card.GetTalent().id == talent.id);
        }
    }

    public void OnConfirm()
    {
        if (selectedTalent != null)
        {
            TalentEngine.Instance.SetCurrentTalent(selectedTalent);
            GameManager.Instance.ChangePhase(GamePhase.HighSchool);
        }
    }

    public void OnRetry()
    {
        var newTalents = TalentEngine.Instance.DrawTalents();
        ShowTalents(newTalents);
    }
}
