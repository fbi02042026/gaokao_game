# 我的高考志愿模拟器 软件说明书

**版本号：V1.0**
**开发单位：苹果核工坊**

---

## 一、软件基本情况

### 1.1 软件名称
我的高考志愿模拟器

### 1.2 软件版本
V1.0

### 1.3 运行环境
- 操作系统：Android、iOS、Windows、macOS
- 运行平台：微信小游戏、抖音小游戏、TapTap
- 引擎版本：团结引擎（Tuanjie Engine）2D URP

### 1.4 主要功能概述

"我的高考志愿模拟器"是一款基于团结引擎开发的人生模拟养成类小游戏。玩家从高三入学开始，经历高中三年学习生活、天赋选择、高考出分、志愿填报、大学生活、职场发展，最终抵达人生结局。游戏融合了天赋系统、性别区分、人格演化、多结局、多周目传承等多种玩法机制。

## 二、软件功能详细说明

### 2.1 性别选择系统

玩家首次进入游戏时，需选择角色性别（男性/女性）。性别选择后，游戏内所有事件插图将根据性别加载对应图片：
- 男性角色：显示男性事件插图
- 女性角色：显示女性事件插图

性别信息存储于玩家存档中（PlayerState.gender），伴随整个游戏流程。存档时性别信息一同保存，确保每次读取存档后性别状态正确。

### 2.2 天赋系统

游戏内置12种天赋，玩家在高三入学时通过三选一方式获得天赋。天赋值贯游戏全流程，影响各阶段属性计算：

| 天赋ID | 名称 | 稀有度 | 效果描述 |
|--------|------|--------|---------|
| T01 | 过目不忘 | SSR | 学习效率提升50% |
| T02 | 逻辑思维 | SSR | 理科成绩+20% |
| T03 | 情绪管理 | SR | 考试稳定性+30% |
| T04 | 社交达人 | SR | 解锁隐藏社交事件 |
| T05 | 运动天才 | R | 体育必定满分 |
| T06 | 幸运星 | SSR | 关键时刻成功率+40% |
| T07 | 速学者 | R | 理解力+30% |
| T08 | 时间操控 | SR | 每日学习时间+2h |
| T09 | 艺术天赋 | SR | 创造力+20 |
| T10 | 领导力 | SSR | 社交收益+50% |
| T11 | 坚韧不拔 | SR | 逆境属性损失减半 |
| T12 | 全面发展 | R | 所有属性均衡增长 |

天赋效果在高中阶段影响属性增长曲线，在高考阶段影响最终成绩计算，在大学和人生阶段影响事件结果和收入水平。

### 2.3 高中三年阶段

玩家进入高三后，通过选择事件选项来塑造角色属性。游戏包含20+个高中事件，覆盖以下类型：
- 学习类（月考、期末复习、模拟考试）
- 情感类（收到情书、与同学吵架）
- 校园类（运动会、班级篮球赛、志愿者活动）
- 家庭类（家长会、家里出状况）
- 关键抉择类（文理分科、考前焦虑）

每个事件提供多个选项，各选项对智力(Intellect)、心态(Mental)、社交(Social)、健康(Health)四项属性产生不同影响。游戏通过时间线展示高三三年六个学期的进程。

### 2.4 高考出分阶段

高三结束进入高考阶段。系统根据玩家三年积累的属性值（智力、心态）、天赋加成和省份因素，计算最终高考成绩。

成绩计算引擎(TalentEngine)综合以下因素：
- 智力基础分（0-100映射到0-300）
- 心态波动分（心态影响考试稳定性）
- 天赋加成（不同天赋有不同加成系数）
- 省份难度系数（各省录取分数线不同）

最终成绩显示采用动画效果，从0分滚动到最终分数。

### 2.5 志愿填报阶段

高考出分后进入志愿填报阶段。系统提供全国44所高校数据，涵盖：
- 985工程高校（清华、北大、复旦、上海交大等）
- 211工程高校（北京邮电、中央财经等）
- 双一流高校
- 普通本科院校

每所高校包含：id、name、tags(985/211标签)、province(所属省份)、type(综合/理工/师范/财经等)、rank(排名)、features(特色)、hot_majors(热门专业)、scores(各省各年录取分数线)。

志愿填报采用三层梯度策略：
1. 冲一冲（高于考生成绩5-15分的院校）
2. 稳一稳（等于或略低于考生成绩的院校）
3. 保一保（低于考生成绩10分以上的院校）

录取结果根据考生成绩与各院校分数线匹配计算。

### 2.6 大学四年阶段

进入大学后，根据考生录取的专业方向，加载对应的事件集：
- 计算机方向：加载college_cs.json（5个事件）
- 法学方向：加载college_law.json（4个事件）
- 医学方向：加载college_med.json（5个事件）
- 商科方向：加载college_biz.json（4个事件）

大学四年期间，玩家需要做出毕业方向选择：
- 考研深造
- 直接就业
- 出国留学
- 创业

不同选择将影响人生阶段的职业路径和收入水平。

### 2.7 人生阶段

毕业方向选择后进入人生阶段。根据毕业选择，人生事件集将有所不同。玩家将在22-30岁期间经历多个关键人生事件，每个事件提供3个选项，影响：
- 月收入变化
- 职业发展
- 家庭状态
- 满意度评分

### 2.8 结局与多周目传承

游戏设置多种结局类型，根据月收入和满意度综合判定：
- 完美结局（月入≥30k且满意度≥4）
- 良好结局（月入≥15k且满意度≥3）
- 苦涩结局（满意度≤2）
- 普通结局（其他情况）

多周目传承机制允许玩家将当前人生的属性加成（按10-15%比例）遗传给下一代，并解锁既视感(Déjà Vu)能力：
- 既视感Lv1：关键选项有微弱提示
- 既视感Lv2：部分事件结果预览
- 既视感Lv3：完美记忆所有选择
- 既视感Lv4：解锁隐藏剧情线

### 2.9 人格系统

游戏内置8种高考人格类型，根据玩家在高中阶段的选择动态计算：

| 人格ID | 名称 | 图标 | 特征描述 |
|--------|------|------|---------|
| Academic | 学术型 | 📚 | 智力倾向明显 |
| Creative | 创意型 | 🎨 | 创造力突出 |
| Social | 社交型 | 🤝 | 社交能力强 |
| Athletic | 运动型 | 🏃 | 身心健康 |
| Balanced | 均衡型 | ⚖️ | 属性均衡发展 |
| Perfectionist | 完美型 | 💎 | 各项追求极致 |
| Rebel | 反叛型 | 🔥 | 打破常规 |
| LateBloomer | 后劲型 | 🌱 | 越到后面越强 |

人格计算基于Intellect/Mental/Social/Health四个维度进行综合评分，取最高维度确定人格类型。

### 2.10 分享系统

游戏内置三种分享卡片生成功能：
- 人格卡：展示玩家高考人格和特征
- 结局卡：展示人生结局和关键数据
- 冷知识卡：展示与志愿相关的小知识

分享图片自动附带游戏二维码或小程序入口。

### 2.11 广告系统

游戏接入激励视频广告机制：
- 激励视频（8种场景）：增加游戏货币、跳过等待、解锁提示等
- Banner广告：主页底部常驻
- 插屏广告：关卡切换时展示

## 三、数据存储结构

```csharp
{
```

```csharp
    public int currentAge;          // 当前年龄
    public GamePhase currentPhase;  // 当前阶段
```

```csharp
public class GameEvent
{

    public string illustrationId;    // 插图ID（男女区分）

    public TimingContent timingContent;     // 计时内容
    public DialogContent dialogContent;     // 对话内容
    public SerializableOutcome[] outcomeList; // 结果列表
}
```
