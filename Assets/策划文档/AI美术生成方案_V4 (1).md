## 一、风格定义与配色体系

### 1.1 风格定位

关键词：2D手绘卡通 / Q版软萌 / 马卡龙暖色 / 休闲小游戏

参考：肥鹅健身房（2D手绘卡通）、恐龙多多（Q版萌系）、抓大鹅（可爱卡通风）

**核心特征：**
- 2D手绘卡通，不是3D渲染，不是果冻塑胶
- Q版人物：圆脸+大眼+腮红+小嘴，头身比约1:1.5
- 扁平涂色+柔和边缘，可有轻微手绘笔触感
- 暖色马卡龙系：奶油白/柔粉/蜜桃橙/薄荷绿/天蓝/暖黄
- 粗圆描边（角色2-3px黑色圆角描边），UI元素细线或无描边
- 圆角大按钮，软糖质感（渐变+微高光，但不是3D）
- 场景可用2.5D等距视角（类似肥鹅健身房的房间视角）
- 装饰元素：星光、爱心、纸屑、小花、小云朵
- 竖版9:16手机屏

**高中阶段→大学/人生阶段属性过渡：** 进度条底板和填充条复用同一套素材，属性名和图标由代码切换，色值不变。

**Prompt参考（填充条）：**
> 2D Q版软萌游戏进度条填充条，[颜色]渐变，圆角矩形，软糖质感（顶部略浅底部略深），右端点缀[装饰]小图标，马卡龙暖色休闲小游戏风格，透明底，9-patch可拉伸（左右圆角+渐变，中间干净区域可拉伸）

- 填充条设为Image Type=Sliced，代码控制fillAmount属性实现动态百分比

- 压力条特殊处理：数值越高压力越大，UI上可考虑反色警示（压力>80时填充条变红）

def process_image(f, prefix, size):
    """处理单张图片"""
    img = Image.open(f"raw/{f}").convert("RGBA")
    
    # 去除白边（自动裁切空白区域）
    bbox = img.getbbox()
    if bbox:
        img = img.crop(bbox)
    
    # 正方形图片居中裁切
    if size[0] == size[1]:
        w, h = img.size
        s = min(w, h)
        img = img.crop(((w-s)//2, (h-s)//2, (w+s)//2, (h+s)//2))
    
    # 缩放到目标尺寸
    img = img.resize(size, Image.LANCZOS)
    
    # 根据类型保存
    if prefix == "bg_":
        # 背景图转为JPG
        rgb = Image.new("RGB", img.size, (255, 248, 240))
        rgb.paste(img, mask=img.split()[3])
        rgb.save(f"output/{f.rsplit('.',1)[0]}.jpg", "JPEG", quality=90)
    else:
        img.save(f"output/{f}", "PNG")

def main():
    os.makedirs("output", exist_ok=True)
    os.makedirs("raw", exist_ok=True)
    
    for f in os.listdir("raw/"):
        if not f.startswith("."):
            # 匹配前缀
            for prefix, size in SIZE_MAP.items():
                if f.startswith(prefix):
                    process_image(f, prefix, size)
                    break

if __name__ == "__main__":
    main()
