## 2.2 合成系统

定位：辅助玩法，为高中阶段增加趣味性，同时作为影响高考分数的重要途径。

设计原则：

• 操作简单：两个相同物品拖拽合成

• 反馈即时：合成动画+音效+属性预览

• 策略深度：不同物品组合产生不同效果

• 广告辅助：可看广告刷新物品或获得稀有物品

详细设计见"4.1 物品/合成系统"

## 2.3 剧情选择系统

定位：核心玩法，承载决策体验的灵魂系统。

设计原则：

• 每个选择都有代价：没有完美选项，只有适合你的选项

• 选择影响命运：重大选择会改变人生轨迹

• 后悔也是体验：不同选择解锁不同剧情线

• 专业差异：不同专业有专属剧情事件

详细设计见"3. 游戏流程"各阶段

## 2.4 养成系统

定位：数值支撑，为所有系统提供成长反馈。

设计原则：

• 双阶段四维属性：智力/心理/社交/健康（高中）→ 元气/压力/朋友/金钱（大学）

• 属性来源多元：合成物品、剧情选择、日常事件均可影响

• 属性决定命运：属性值决定可报考院校、可触发事件

• 可视化成长：属性面板直观展示成长轨迹

详细设计见"4.3 属性系统"

## 2.5 学长学姐支招系统（核心差异化功能）

定位：差异化卖点，用真实声音辅助志愿决策。

设计原则：

• 真实来源：来自600页学长学姐专业解读资源

• 情感连接：NPC对话形式，比数据更有温度

• 实用价值：真实的就读体验、就业情况、发展建议

• 广告变现：核心广告位之一

详细设计见"2.5.1 学长学姐支招功能"

## 3.2 高中阶段设计（⚡紧凑节奏：~30分钟当天玩完）

### 3.2.1 阶段定位

核心目标：

• 养成+合成+事件，体验完整但不拖

• 选科决策是高中阶段最重的决策

• 天赋抽卡系统增加策略深度和重玩动力

• 为高考后的慢节奏正戏做好铺垫

时间跨度：高一入学 → 高三毕业（~30分钟真实时间，当天一次玩完）

### 3.2.2 天赋系统（贯穿高考→专业→职业的线索）

设计理念：天赋不是单纯的数值buff，而是贯穿整局游戏的性格标签。同一个天赋在高中、高考、选专业、大学、职场各阶段都有不同的表现和影响，形成"这个人的天赋决定了他的人生走向"的因果感。

开局流程：随机抽取3个天赋卡 → 3选1保留 → 可看广告重新抽3张

既视感分级（根据前世记忆天赋等级）：

|天赋等级|提示内容|示例|
|---|---|---|
|无天赋|无提示|正常体验|
|🌀 模糊预感|只提示"上次选了什么"|"上次好像选了疯狂刷题……"|
|🔮 隐约记得|提示上次选择的结果方向|"上次选了疯狂刷题，考得还行但累坏了……"|
|👁️ 前世回响|提示结果方向+属性变化|"上次选了疯狂刷题，智力+但心理-，后来……"|
|⚡ 直觉闪现|最佳选项微微发光|选项B微微发金光（但不说明为什么）|




既视感触发规则：

• 只在相同事件触发，不同事件不触发

• 如果上次事件因不同前置条件导致不同选项，既视感不会触发

• 既视感只提供模糊提示，不剧透具体结果，保留选择悬念

• 相同选项第二次选，既视感提示会升级："又选了这个？结果你应该知道了……"

• 全新事件（前世未遇到）不会触发既视感，保持新鲜感

• 既视感与天赋联动：如果本次天赋不同，既视感会提示"上次的天赋是XX，这次你是YY，结果可能不同"

### 4.4.4 前世记忆天赋叠加

每通关一次，下一代额外获得1个"前世记忆"天赋（随机抽取），可叠加：

|天赋名称|效果|稀有度|
|---|---|---|
|🌀 模糊预感|遇到上次选过的选项时，浮现"上次好像选了这个..."|常见|
|🔮 隐约记得|既视感提示更具体："上次选了这个，好像结果不太好..."|稀有|
|👁️ 前世回响|既视感提示包含结果方向："上次选了这个，后来..."|史诗|
|⚡ 直觉闪现|既视感事件中，最佳选项会微微发光|史诗（需通关3次）|




叠加规则：第二代有1个前世记忆天赋，第三代有2个，第四代有3个……记忆越来越清晰。

### 4.4.6 既视感的情感价值

既视感不仅仅是游戏机制，更是一种情感体验：

• 好奇驱动："如果这次选另一个呢？" → 不是速通跳过，而是认真重走

• 补偿心理："上次选了这个后悔了，这次换一个" → 选择更有分量

• 成长感："第三次了，这次我一定选对" → 玩家在成长，不是角色

• 命运感："原来不同的选择真的会改变一切" → 呼应游戏主题

# 6. 数值设定

## 6.1 分数计算系统

### 6.1.1 高考分数构成

高考总分 = 基础分 + 物品加成 + 随机波动

基础分：400分（确保不白卷）
物品加成：0~350分（由合成系统决定）
随机波动：-20~+20分（增加不确定性）
─────────────────────────────────────
总分范围：380~770分（理论上限，750为实用上限）


## 6.2 录取概率算法

### 6.2.1 基础概率计算

基础概率 = F(分数差, 招生计划, 报考热度)

分数差 = 考生分数 - 院校投档线

概率区间：
├─ 分数差 > +30分 → 概率 95%+
├─ 分数差 +10~30  → 概率 80-95%
├─ 分数差 -10~+10 → 概率 50-80%
├─ 分数差 -30~-10 → 概率 20-50%
└─ 分数差 < -30分 → 概率 <20%

## 14.2 MBTI式高考人格系统

### 8种高考人格

根据玩家在游戏中的所有选择，计算最终人格标签（类似MBTI）：

|人格|图标|标签语|触发倾向|传播点|
|---|---|---|---|---|
|🔬 理性探索者|🔬|"用数据说话，用选择定义人生"|高智力+高策略评分|学霸认同|
|🎨 自由创作者|🎨|"不走寻常路，活出自己的颜色"|高心理+选艺术/文学|文艺青年共鸣|
|💼 实用主义者|💼|"理想很美，但我选靠谱"|选就业好专业+薪资优先|务实派认同|
|🔥 逆袭斗士|🔥|"人生没有定局，我来翻盘"|低分开局+全选冲刺|热血感染力|
|🌊 佛系随缘|🌊|"该来的总会来，急什么"|多次选轻松选项|佛系青年共鸣|
|❤️ 情感驱动|❤️|"人心比分数重要"|高社交+情感选项|温暖感染力|
|🏔️ 稳扎稳打|🏔️|"不求最高，但求最稳"|全选保底+求稳|家长最认同|
|🌟 理想主义者|🌟|"为热爱而活，不将就"|选冷门专业+追梦|理想主义共鸣|




### 人格计算逻辑

基于玩家最终属性权重得分：

每个人格有 statWeights = { intellect: x, mental: y, social: z, health: w }
人格得分 = intellect × x + mental × y + social × z + health × w
得分最高的人格 = 玩家的高考人格


同时计算3条展示指数（0-100%）：理性指数、冲动指数、冒险指数
