# P3 学长学姐评价系统 — 实施计划

## 目标
实现V3.0文档定义的学长学姐支招系统，作为核心差异化功能集成到志愿填报(Zhiyuan)阶段。

## 设计依据
- 数据来源：600页学长学姐专业解读资源
- 交互入口：院校卡片上的"学长说"按钮
- 解锁方式：免费试听1条 + 广告解锁更多
- 5种NPC人设：学霸型学长/学姐型学姐/奋斗型学长/佛系型学姐/转行型学长
- 内容经包装改写，不照搬原文

---

## 步骤1: 创建评价数据模型 + JSON数据文件

### 文件: `Assets/Resources/Data/senior_evals.json`
- 数据量：文档中有20+条评价（计算机、电子信息、医学、法学、经管、其他），先完整录入
- 每条评价结构：
  ```json
  {
    "evalId": "EVL_001_01",
    "majorId": "计算机科学与技术",
    "reviewer": {
      "identity": "某985计算机系学长",
      "collegeType": "985",
      "yearsAfterGraduation": "5年",
      "currentStatus": "某大厂高级工程师",
      "npcType": "学霸型学长"
    },
    "content": "评价原文...",
    "tags": ["高薪", "加班多"],
    "adLocked": false,
    "priority": 1
  }
  ```

### 文件: `Assets/Scripts/Models/SeniorEval.cs`（新建）
- C# 模型类：
  - `SeniorEval` — 单条评价（evalId, majorId, content, tags, adLocked, priority）
  - `ReviewerInfo` — 评价者信息（identity, collegeType, npcType, currentStatus）
  - `SeniorEvalData` — 包装类，含 `List<SeniorEval>` 用于 JSON 反序列化

---

## 步骤2: 创建 SeniorEvalEngine 引擎

### 文件: `Assets/Scripts/Engine/SeniorEvalEngine.cs`（新建）
- 单例模式（DontDestroyOnLoad）
- Awake 时从 `Resources/Data/senior_evals.json` 自动加载
- 核心方法：
  - `GetEvalsByMajor(string majorId)` → 返回某专业的所有评价
  - `GetFreeTrial(string majorId)` → 返回第一条（adLocked=false）的评价
  - `IsUnlocked(string evalId)` → 检查评价是否已解锁
  - `UnlockEval(string evalId)` → 标记为已解锁（持久化）
  - `UnlockViaAd(string evalId)` → 广告回调后的解锁
- 解锁状态管理：通过 `PlayerPrefs` 存储（小游戏兼容）

## 步骤4: 改造 ZhiyuanUI — 添加"学长说"入口

### 文件: `Assets/Scripts/UI/ZhiyuanUI.cs`（修改）
- 在 `BuildCollegeCard()` 中，为每张院校卡片添加"学长说"按钮
- 按钮点击 → 弹出 SeniorEvalPanel
- 按钮显示状态：

  - 已有免费试听 + 有广告锁定评价 → 显示"🎬查看全部"
- 新增 UI 引用：
  ```csharp
  [SerializeField] private GameObject seniorEvalPanelPrefab;
  ```

## 步骤5: 创建 SeniorEvalPanel UI 脚本 + 预制体生成

### 文件: `Assets/Scripts/UI/SeniorEvalPanel.cs`（新建）
- 面板结构：
  1. **标题栏** — 专业名称 + 关闭按钮
  2. **免费评价区** — 显示第一条（adLocked=false）的评价
     - NPC头像占位 + 身份标签
     - 评价原文滚动区
     - 标签 chips（#高薪 #加班多）
  3. **广告解锁区** — 更多评价列表
     - 每条预览前30字 + 🔒图标
     - "🎬观看广告解锁" 按钮 → `PlatformManager.ShowRewardedVideo()`
  4. **5种NPC类型卡片** — 底部横向滚动，展示不同人设的评价者信息
- 动画：
  - 面板从底部滑入
  - 评价文字逐行显示（typewriter effect）
- 多周目既视感：
  - 如果 `playerState.hasPastLifeMemory` 且该评价之前看过 → 显示"💭似曾相识"标签
