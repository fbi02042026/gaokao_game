## 1. 软件基本情况

### 1.1 软件名称
我的高考志愿模拟器软件 V1.0

### 1.2 软件版本
V1.0

### 1.3 运行环境
- 操作系统：Android、iOS、Windows、macOS
- 运行平台：微信小游戏、抖音小游戏、TapTap
- 引擎版本：团结引擎（Tuanjie Engine）2D URP

### 1.4 主要功能概述

"我的高考志愿模拟器软件 V1.0"是一款基于团结引擎开发的人生模拟养成类小游戏。玩家从高三入学开始，经历高中三年学习生活、天赋选择、高考出分、志愿填报、大学生活、职场发展，最终抵达人生结局。游戏融合了天赋系统、性别区分、人格演化、多结局、多周目传承等多种玩法机制。

## 2. 软件功能详细说明

### 2.1 性别选择系统

玩家首次进入游戏时，需选择角色性别（男性/女性）。性别选择后，游戏内所有事件插图将根据性别加载对应图片：
- 男性角色：显示男性事件插图
- 女性角色：显示女性事件插图

性别信息存储于玩家存档中（PlayerState.gender），伴随整个游戏流程。存档时性别信息一同保存，确保每次读取存档后性别状态正确。

### 2.2 天赋系统

游戏内置12种天赋，玩家在高三入学时通过三选一方式获得天赋。天赋值贯游戏全流程，影响各阶段属性计算：

| 天赋ID | 名称 | 稀有度 | 效果描述 |
|--------|------|--------|---------|
| T01 | 过目不忘 | SSR | 学习效率提升50% |
| T02 | 逻辑思维 | SSR | 理科成绩+20% |
| T03 | 情绪管理 | SR | 考试稳定性+30% |
| T04 | 社交达人 | SR | 解锁隐藏社交事件 |
| T05 | 运动天才 | R | 体育必定满分 |
| T06 | 幸运星 | SSR | 关键时刻成功率+40% |
| T07 | 速学者 | R | 理解力+30% |
| T08 | 时间操控 | SR | 每日学习时间+2h |
| T09 | 艺术天赋 | SR | 创造力+20 |
| T10 | 领导力 | SSR | 社交收益+50% |
| T11 | 坚韧不拔 | SR | 逆境属性损失减半 |
| T12 | 全面发展 | R | 所有属性均衡增长 |

天赋效果在高中阶段影响属性增长曲线，在高考阶段影响最终成绩计算，在大学和人生阶段影响事件结果和收入水平。

### 2.3 高中三年阶段

玩家进入高三后，通过选择事件选项来塑造角色属性。游戏包含20+个高中事件，覆盖以下类型：
- 学习类（月考、期末复习、模拟考试）
- 情感类（收到情书、与同学吵架）
- 校园类（运动会、班级篮球赛、志愿者活动）
- 家庭类（家长会、家里出状况）
- 关键抉择类（文理分科、考前焦虑）

每个事件提供多个选项，各选项对智力(Intellect)、心态(Mental)、社交(Social)、健康(Health)四项属性产生不同影响。游戏通过时间线展示高三三年六个学期的进程。

### 2.4 高考出分阶段

高三结束进入高考阶段。系统根据玩家三年积累的属性值（智力、心态）、天赋加成和省份因素，计算最终高考成绩。

成绩计算引擎(TalentEngine)综合以下因素：
- 智力基础分（0-100映射到0-300）
- 心态波动分（心态影响考试稳定性）
- 天赋加成（不同天赋有不同加成系数）
- 省份难度系数（各省录取分数线不同）

最终成绩显示采用动画效果，从0分滚动到最终分数。

### 2.5 志愿填报阶段

高考出分后进入志愿填报阶段。系统提供全国44所高校数据，涵盖：
- 985工程高校（清华、北大、复旦、上海交大等）
- 211工程高校（北京邮电、中央财经等）
- 双一流高校
- 普通本科院校

每所高校包含：id、name、tags(985/211标签)、province(所属省份)、type(综合/理工/师范/财经等)、rank(排名)、features(特色)、hot_majors(热门专业)、scores(各省各年录取分数线)。

志愿填报采用三层梯度策略：
1. 冲一冲（高于考生成绩5-15分的院校）
2. 稳一稳（等于或略低于考生成绩的院校）
3. 保一保（低于考生成绩10分以上的院校）

录取结果根据考生成绩与各院校分数线匹配计算。

### 2.6 大学四年阶段

进入大学后，根据考生录取的专业方向，加载对应的事件集：
- 计算机方向：加载college_cs.json（5个事件）
- 法学方向：加载college_law.json（4个事件）
- 医学方向：加载college_med.json（5个事件）
- 商科方向：加载college_biz.json（4个事件）

大学四年期间，玩家需要做出毕业方向选择：
- 考研深造
- 直接就业
- 出国留学
- 创业

不同选择将影响人生阶段的职业路径和收入水平。

### 2.7 人生阶段

毕业方向选择后进入人生阶段。根据毕业选择，人生事件集将有所不同。玩家将在22-30岁期间经历多个关键人生事件，每个事件提供3个选项，影响：
- 月收入变化
- 职业发展
- 家庭状态
- 满意度评分

### 2.8 结局与多周目传承

游戏设置多种结局类型，根据月收入和满意度综合判定：
- 完美结局（月入≥30k且满意度≥4）
- 良好结局（月入≥15k且满意度≥3）
- 苦涩结局（满意度≤2）
- 普通结局（其他情况）

多周目传承机制允许玩家将当前人生的属性加成（按10-15%比例）遗传给下一代，并解锁既视感(Déjà Vu)能力：
- 既视感Lv1：关键选项有微弱提示
- 既视感Lv2：部分事件结果预览
- 既视感Lv3：完美记忆所有选择
- 既视感Lv4：解锁隐藏剧情线

### 2.9 人格系统

游戏内置8种高考人格类型，根据玩家在高中阶段的选择动态计算：

| 人格ID | 名称 | 图标 | 特征描述 |
|--------|------|------|---------|
| Academic | 学术型 | 📚 | 智力倾向明显 |
| Creative | 创意型 | 🎨 | 创造力突出 |
| Social | 社交型 | 🤝 | 社交能力强 |
| Athletic | 运动型 | 🏃 | 身心健康 |
| Balanced | 均衡型 | ⚖️ | 属性均衡发展 |
| Perfectionist | 完美型 | 💎 | 各项追求极致 |
| Rebel | 反叛型 | 🔥 | 打破常规 |
| LateBloomer | 后劲型 | 🌱 | 越到后面越强 |

人格计算基于Intellect/Mental/Social/Health四个维度进行综合评分，取最高维度确定人格类型。

### 2.10 分享系统

游戏内置三种分享卡片生成功能：
- 人格卡：展示玩家高考人格和特征
- 结局卡：展示人生结局和关键数据
- 冷知识卡：展示与志愿相关的小知识

分享图片自动附带游戏二维码或小程序入口。

### 2.11 广告系统

游戏接入激励视频广告机制：
- 激励视频（8种场景）：增加游戏货币、跳过等待、解锁提示等
- Banner广告：主页底部常驻
- 插屏广告：关卡切换时展示

## 一、系统架构概述

### 1.1 整体架构

本软件采用Unity/Tuanjie Engine 2D URP架构，使用C#进行开发，遵循以下分层架构设计：

```
┌─────────────────────────────────────────────┐
│              UI Layer (界面层)               │
│  HomeUI / GenderSelectUI / HighSchoolUI     │
│  CollegeUI / LifeUI / ResultUI              │
├─────────────────────────────────────────────┤
│           Manager Layer (管理器层)           │
│  GameManager / GameStateManager / SaveManager│
│  AdManager / ShareManager / PlatformManager │
├─────────────────────────────────────────────┤
│            Engine Layer (引擎层)             │
│  EventEngine / TalentEngine / ScoreEngine   │
│  PersonalityEngine / DejaVuEngine            │
│  InheritEngine / MergeEngine / LifeEngine   │
├─────────────────────────────────────────────┤
│             Data Layer (数据层)             │
│  DataLoader / ResourceHelper                │
├─────────────────────────────────────────────┤
│            Model Layer (模型层)              │
│  PlayerState / GameEvent / Talent           │
│  College / Major / Province / NPC            │
└─────────────────────────────────────────────┘
```

### 1.2 核心设计模式

- **单例模式(Singleton)**：所有Manager和Engine类使用Instance单例模式，确保全局唯一
- **观察者模式(Observer)**：GameManager通过event实现状态变更通知
- **工厂模式**：DataLoader作为数据工厂统一创建各类游戏数据对象
- **策略模式**：事件系统支持多种交互策略（choice/merge/sort/slider/timing/dialog）

## 二、核心模块设计

### 2.1 游戏管理器（GameManager）

游戏总控制器，负责游戏生命周期管理。

**主要职责：**
- 管理游戏全局状态（isGameRunning, isPaused）
- 协调各Engine和Manager之间的通信
- 处理场景切换逻辑
- 管理玩家数据读写

**核心方法：**
- `StartNewGame()`：初始化新游戏流程
- `ContinueGame()`：从存档恢复游戏
- `ChangePhase(GamePhase)`：阶段切换
- `SaveGame(slotIndex)`：保存游戏数据
- `CalculateGaokaoScore()`：调用ScoreEngine计算高考成绩
- `StartNewGamePlus()`：二周目新游戏创建

### 2.2 游戏状态管理器（GameStateManager）

独立于GameManager的状态管理组件，生命周期贯穿所有场景。

**主要职责：**
- 维护GameSaveData数据结构
- 提供QuickSave/QuickLoad快速存取接口
- 管理玩家性别（Gender）信息
- 协调存档与引擎间的数据同步

**与GameManager的区别：**
- GameManager专注游戏流程控制
- GameStateManager专注数据持久化

### 2.3 事件引擎（EventEngine）

驱动游戏进程的核心引擎，负责事件的获取、选择和结算。

**核心流程：**
```
获取事件 → 检查触发条件 → 检查既视感 → 展示事件
    ↓
玩家交互选择 → 调用ResolveEvent计算结果 → 更新属性
    ↓
记录事件完成 → 获取下一事件 → 循环
```

**核心方法：**
- `SetPlayerState(PlayerState)`：注入玩家状态
- `GetNextEvent()`：获取下一事件
- `ResolveEvent(eventId, choiceId)`：结算事件结果
- `GetWeightedRandom()`：加权随机选择

### 2.4 天赋引擎（TalentEngine）

管理12种天赋的抽卡、选择和效果计算。

**核心方法：**
- `DrawTalents(int count)`：抽卡获取候选天赋
- `SelectTalent(string talentId)`：玩家选择天赋
- `GetCurrentTalent()`：获取当前天赋
- `ApplyTalentEffect(PlayerState)`：应用天赋效果
- `GetTalentBonus(string statName)`：获取天赋对特定属性的加成

### 2.5 成绩计算引擎（ScoreEngine）

根据玩家属性和天赋计算高考成绩。

**计算公式：**
```
最终成绩 = 智力基础分 × 0.7 + 心态影响分 × 0.2 + 天赋加成 + 省份难度调整
```

**核心方法：**
- `CalculateScore(PlayerState, string talentId)`：主计算入口
- `GetSubjectScore(subject, baseStat)`：计算单科成绩
- `ApplyProvinceModifier(score, province)`：应用省份难度系数

### 2.6 人格引擎（PersonalityEngine）

根据玩家属性计算高考人格类型。

**人格判定规则：**
- 取Intellect/Mental/Social/Health四个维度的最高值
- 若存在多个维度相等，取优先级最高的维度对应人格
- 人格类型决定游戏内部分对话和结局倾向

### 2.7 既视感引擎（DejaVuEngine）

二周目记忆系统，提供前世记忆和选择提示。

**既视感等级（L1-L4）：**
- L1：微弱直觉，仅在关键选项时显示模糊提示
- L2：记忆片段，显示选项可能导致的属性变化方向
- L3：清晰记忆，显示每个选项的历史结果
- L4：完美直觉，解锁隐藏剧情选项

### 2.8 传承引擎（InheritEngine）

多代传承系统，将当前人生的属性遗传给下一代。

**传承内容：**
- 属性加成（Intellect/Mental/Social/Health各传承10-15%）
- 记忆天赋（既视感等级）
- 校友院校（解锁特定招生优惠）

```csharp
public class EventEngine : MonoBehaviour
{
    private List<GameEvent> allEvents = new List<GameEvent>();
    private List<string> completedEventIds = new List<string>();
    private PlayerState playerState;
    private System.Random rng;

    public void SetPlayerState(PlayerState state)
    {
        playerState = state;
        if (state != null)
        {
            completedEventIds = new List<string>(state.completedEvents);
            rng = new System.Random();
        }
    }

    public IEnumerator LoadEventsFromStreamingAssets(string fileName)
    {
        string path = System.IO.Path.Combine(Application.streamingAssetsPath, "Events", $"{fileName}.json");
        UnityWebRequest request = UnityWebRequest.Get(path);
        yield return request.SendWebRequest();

        if (request.result == UnityWebRequest.Result.Success)
        {
            string json = request.downloadHandler.text;
            var wrapper = JsonHelper.GetJsonList<GameEvent>(json);
            allEvents = wrapper ?? new List<GameEvent>();
        }
    }

    public GameEvent GetNextEvent()
    {
        if (allEvents.Count == 0) return null;

        var candidate = allEvents
            .Where(e => !completedEventIds.Contains(e.id))
            .Where(e => MeetsCondition(e))
            .ToList();

        if (candidate.Count == 0) return null;

        return candidate[rng.Next(candidate.Count)];
    }

    public EventOutcome ResolveEvent(string eventId, string choiceId)
    {
        var evt = allEvents.FirstOrDefault(e => e.id == eventId);
        if (evt == null) return null;

        completedEventIds.Add(eventId);
        if (playerState != null)
        {
            if (!playerState.completedEvents.Contains(eventId))
                playerState.completedEvents.Add(eventId);
            playerState.RecordChoice(eventId, choiceId);
        }

        var outcomes = evt.GetOutcomeDict();
        if (outcomes.TryGetValue(choiceId, out var outcome))
        {
            ApplyOutcome(outcome);
            return outcome;
        }

        return null;
    }
}
```

```csharp
public class TalentEngine : MonoBehaviour
{
    public static TalentEngine Instance { get; private set; }
    private List<Talent> allTalents = new List<Talent>();
    private Talent currentTalent;
    private System.Random rng = new System.Random();

    void Awake()
    {
        if (Instance == null) { Instance = this; DontDestroyOnLoad(gameObject); }
        else { Destroy(gameObject); }
    }

    public void LoadTalents(List<Talent> talents)
    {
        allTalents = talents ?? new List<Talent>();
    }

    public List<Talent> DrawTalents(int count = 3)
    {
        if (allTalents.Count == 0) return new List<Talent>();
        count = Mathf.Min(count, allTalents.Count);

        var candidates = allTalents
            .OrderBy(t => -t.probability)
            .ThenBy(_ => rng.Next())
            .Take(count * 3)
            .ToList();

        return candidates.OrderBy(_ => rng.Next()).Take(count).ToList();
    }

    public void SelectTalent(string talentId)
    {
        currentTalent = allTalents.FirstOrDefault(t => t.id == talentId);
    }

    public Talent GetCurrentTalent() => currentTalent;

    public float GetTalentBonus(string statName)
    {
        if (currentTalent?.effects == null) return 1f;

        foreach (var effect in currentTalent.effects)
        {
            if (effect.type == "stat_multiply" && effect.statKey == statName)
                return 1f + effect.value / 100f;
        }
        return 1f;
    }
}
```

```csharp
public class ScoreEngine : MonoBehaviour
{
    public static ScoreEngine Instance { get; private set; }
    private System.Random rng = new System.Random();

    void Awake()
    {
        if (Instance == null) { Instance = this; DontDestroyOnLoad(gameObject); }
        else { Destroy(gameObject); }
    }

    public int CalculateScore(PlayerState state, string talentId = "")
    {
        float intellectScore = GetSubjectScore("语文", state.intellect)
                             + GetSubjectScore("数学", state.intellect)
                             + GetSubjectScore("英语", state.intellect);

        float scienceScore = GetSubjectScore("物理", state.intellect)
                          + GetSubjectScore("化学", state.intellect)
                          + GetSubjectScore("生物", state.intellect);

        float liberalScore = GetSubjectScore("政治", state.intellect)
                           + GetSubjectScore("历史", state.intellect)
                           + GetSubjectScore("地理", state.intellect);

        bool isScience = state.grade >= 3;
        float baseScore = isScience ? scienceScore : liberalScore;
        float totalBase = intellectScore + baseScore;

        float mentalBonus = (state.mental - 50) * 1.5f;
        float mentalVariation = rng.Next(-15, 16);

        float talentBonus = GetTalentBonus(talentId);

        float finalScore = totalBase + mentalBonus + mentalVariation + talentBonus;
        return Mathf.RoundToInt(Mathf.Clamp(finalScore, 0, 750));
    }

    public int GetSubjectScore(string subject, int baseStat)
    {
        float baseScore = baseStat * 0.7f + rng.Next(-5, 6);
        return Mathf.RoundToInt(Mathf.Clamp(baseScore, 0, 150));
    }
}
```

```csharp
public class PersonalityEngine : MonoBehaviour

    public static PersonalityEngine Instance { get; private set; }
    private List<Personality> allPersonalities = new List<Personality>();

    void Awake()
    {
        if (Instance == null) { Instance = this; DontDestroyOnLoad(gameObject); }
        else { Destroy(gameObject); }

        allPersonalities = personalities ?? new List<Personality>();
    }

    public PersonalityResult GetPersonalityScores(PlayerState state)
    {
        if (state == null) return null;

        int intellect = state.intellect;
        int mental = state.mental;
        int social = state.social;
        int health = state.health;

        int maxStat = Mathf.Max(intellect, mental, social, health);
        PersonalityType dominant = PersonalityType.Balanced;

        if (maxStat == intellect) dominant = PersonalityType.Academic;
        else if (maxStat == mental) dominant = PersonalityType.Perfectionist;
        else if (maxStat == social) dominant = PersonalityType.Social;
        else if (maxStat == health) dominant = PersonalityType.Athletic;

        var personality = allPersonalities.FirstOrDefault(p => 
            p.id == dominant.ToString().ToLower()) ?? allPersonalities.FirstOrDefault();

        return new PersonalityResult
        {
            personality = personality,
            rational = Mathf.Clamp((intellect * 0.6f + mental * 0.3f + social * 0.1f), 0, 100),
            impulsive = Mathf.Clamp((mental * 0.5f + social * 0.3f + health * 0.2f), 0, 100),
            adventurous = Mathf.Clamp((social * 0.4f + health * 0.4f + mental * 0.2f), 0, 100)
        };
    }
}
```

```csharp
            genderSelectUI.Show();
        }
        else
        {
            genderSelected = true;
            if (mainMenuPanel != null)
                mainMenuPanel.SetActive(true);
        }
    }

    private void OnGenderSelected(Gender gender)
    {
        genderSelected = true;
        if (genderSelectUI != null)
            genderSelectUI.OnGenderSelected -= OnGenderSelected;

        if (mainMenuPanel != null)
            mainMenuPanel.SetActive(true);
    }

    void OnStartGame()
    {
        if (!genderSelected)
            return;

        if (GameStateManager.Instance != null)
            GameStateManager.Instance.NewGame();

        UnityEngine.SceneManagement.SceneManager.LoadScene("HighSchool");
    }

    void OnContinueGame()
    {
        if (GameStateManager.Instance != null)
            GameStateManager.Instance.QuickLoad();
    }
}
```

```csharp
public class GenderSelectUI : MonoBehaviour
{
    [SerializeField] private GameObject panelRoot;
    [SerializeField] private Button boyBtn;
    [SerializeField] private Button girlBtn;
    [SerializeField] private Image boyImage;
    [SerializeField] private Image girlImage;

    public System.Action<Gender> OnGenderSelected;

    void Start()
    {
        if (boyBtn != null) boyBtn.onClick.AddListener(() => SelectGender(Gender.Male));
        if (girlBtn != null) girlBtn.onClick.AddListener(() => SelectGender(Gender.Female));

        LoadImages();
    }

    private void LoadImages()
    {
        if (boyImage != null)
        {
            var sprite = ResourceHelper.LoadGenderBoy();
            if (sprite != null) boyImage.sprite = sprite;
        }
        if (girlImage != null)
        {
            var sprite = ResourceHelper.LoadGenderGirl();
            if (sprite != null) girlImage.sprite = sprite;
        }
    }

    public void Show() { if (panelRoot != null) panelRoot.SetActive(true); }
    public void Hide() { if (panelRoot != null) panelRoot.SetActive(false); }

    private void SelectGender(Gender gender)
    {
        PlayerState state = null;
        if (GameManager.Instance != null)
            state = GameManager.Instance.CurrentPlayer;
        if (state == null && SaveManager.Instance != null)
            state = SaveManager.Instance.GetCurrentPlayer();
        if (state == null)
            state = new PlayerState();

        state.gender = gender;

        if (SaveManager.Instance != null)
            SaveManager.Instance.SetCurrentPlayer(state);

        OnGenderSelected?.Invoke(gender);
        Hide();
    }
}
```

## 六、GameManager - 总控制器

```csharp
public class GameManager : MonoBehaviour
{
    public static GameManager Instance { get; private set; }

    private PlayerState currentPlayer;
    private GamePhase currentPhase;
    private bool isGameRunning;
    private bool isPaused;

    private List<string> gameSceneNames = new List<string>
    {
        "Home", "Province", "HighSchool", "Event",
        "Gaokao", "Zhiyuan", "College", "Life", "Result"
    };

    public event System.Action<PlayerState> OnPlayerStateChanged;
    public event System.Action<GamePhase> OnPhaseChanged;
    public event System.Action OnGameStarted;
    public event System.Action OnGameEnded;

    public PlayerState CurrentPlayer => currentPlayer;
    public GamePhase CurrentPhase => currentPhase;
    public bool IsGameRunning => isGameRunning;
    public bool IsPaused => isPaused;

    void Awake()
    {
        if (Instance == null)
        {
            Instance = this;
            DontDestroyOnLoad(gameObject);
        }
        else
        {
            Destroy(gameObject);
            return;
        }

        Initialize();
    }

    private void Initialize()
    {
        currentPhase = GamePhase.Home;
        isGameRunning = false;
        isPaused = false;
    }

    public void StartNewGame()
    {
        Gender savedGender = Gender.Male;
        if (SaveManager.Instance != null)
        {
            var existing = SaveManager.Instance.GetCurrentPlayer();
            if (existing != null) savedGender = existing.gender;
        }

        if (SaveManager.Instance != null)
        {
            SaveManager.Instance.CreateNewGame();
            currentPlayer = SaveManager.Instance.GetCurrentPlayer();
            if (currentPlayer != null) currentPlayer.gender = savedGender;
        }
        else
        {
            currentPlayer = new PlayerState();
        }

        isGameRunning = true;
        currentPhase = GamePhase.Home;

        LoadGameData();

        OnPlayerStateChanged?.Invoke(currentPlayer);
        OnPhaseChanged?.Invoke(currentPhase);
        OnGameStarted?.Invoke();

        LoadScene("Home");
    }

    public void ContinueGame()
    {
        if (SaveManager.Instance != null && SaveManager.Instance.LoadGame())
        {
            currentPlayer = SaveManager.Instance.GetCurrentPlayer();
            isGameRunning = true;

            LoadGameData();

            OnPlayerStateChanged?.Invoke(currentPlayer);
            OnPhaseChanged?.Invoke(currentPlayer.currentPhase);

            LoadScene(currentPlayer.currentPhase.ToString());
        }
    }

    public void ChangePhase(GamePhase newPhase)
    {
        if (currentPhase == newPhase) return;

        currentPhase = newPhase;
        currentPlayer.currentPhase = newPhase;

        OnPhaseChanged?.Invoke(newPhase);
        LoadScene(newPhase.ToString());
    }

    public void LoadScene(string sceneName)
    {
        if (gameSceneNames.Contains(sceneName))
        {
            SceneManager.LoadScene(sceneName);
        }
    }

    public void SaveGame(int slotIndex = 0)
    {
        if (currentPlayer == null)
            return;

        currentPlayer.lastPlayTime = System.DateTime.Now;

        if (SaveManager.Instance != null)
        {
            SaveManager.Instance.SetCurrentPlayer(currentPlayer);
            SaveManager.Instance.SaveGame(slotIndex);
        }
    }

    public int CalculateGaokaoScore()
    {
        if (ScoreEngine.Instance != null)
        {
            string talentId = TalentEngine.Instance?.GetCurrentTalent()?.id ?? "";
            return ScoreEngine.Instance.CalculateScore(currentPlayer, talentId);
        }
        return 400;
    }

    public void StartNewGamePlus()
    {
        Gender savedGender = Gender.Male;
        var existing = SaveManager.Instance?.GetCurrentPlayer();
        if (existing != null) savedGender = existing.gender;

        SaveManager.Instance?.CreateNewGame();
        currentPlayer = SaveManager.Instance?.GetCurrentPlayer() ?? new PlayerState();
        currentPlayer.gender = savedGender;
        currentPlayer.hasPastLifeMemory = true;

        isGameRunning = true;
        ChangePhase(GamePhase.HighSchool);
    }
}
```
