1.4 主要功能概述

"我的高考志愿模拟器软件 V1.0"是一款基于团结引擎开发的人生模拟养成类小游戏。玩家从高三入学开始，经历高中三年学习生活、天赋选择、高考出分、志愿填报、大学生活、职场发展，最终抵达人生结局。游戏融合了天赋系统、性别区分、人格演化、多结局、多周目传承等多种玩法机制。

2. 软件功能详细说明

2.1 性别选择系统

玩家首次进入游戏时，需选择角色性别（男性/女性）。性别选择后，游戏内所有事件插图将根据性别加载对应图片：男性角色显示男性事件插图，女性角色显示女性事件插图。性别信息存储于玩家存档中（PlayerState.gender），伴随整个游戏流程。

2.2 天赋系统

游戏内置12种天赋，玩家在高三入学时通过三选一方式获得天赋。天赋值贯穿游戏全流程，影响各阶段属性计算。

2.3 高中三年阶段

玩家进入高三后，通过选择事件选项来塑造角色属性。游戏包含20+个高中事件，覆盖学习类（月考、期末复习、模拟考试）、情感类（收到情书、与同学吵架）、校园类（运动会、班级篮球赛、志愿者活动）、家庭类（家长会、家里出状况）、关键抉择类（文理分科、考前焦虑）等类型。

2.4 高考出分阶段

高三结束进入高考阶段。系统根据玩家三年积累的属性值（智力、心态）、天赋加成和省份因素，计算最终高考成绩。成绩计算引擎综合智力基础分、心态波动分、天赋加成、省份难度系数等因素。最终成绩显示采用动画效果，从0分滚动到最终分数。

2.5 志愿填报阶段

高考出分后进入志愿填报阶段。系统提供全国44所高校数据，涵盖985工程高校、211工程高校、双一流高校、普通本科院校。志愿填报采用三层梯度策略：冲一冲（高于考生成绩5-15分的院校）、稳一稳（等于或略低于考生成绩的院校）、保一保（低于考生成绩10分以上的院校）。

2.6 大学四年阶段

进入大学后，根据考生录取的专业方向，加载对应的事件集：计算机方向（college_cs.json）、法学方向（college_law.json）、医学方向（college_med.json）、商科方向（college_biz.json）。大学四年期间，玩家需要做出毕业方向选择：考研深造、直接就业、出国留学、创业。

2.7 人生阶段

毕业方向选择后进入人生阶段。根据毕业选择，人生事件集将有所不同。玩家将在22-30岁期间经历多个关键人生事件，每个事件提供3个选项，影响月收入变化、职业发展、家庭状态、满意度评分。

2.8 结局与多周目传承

游戏设置多种结局类型：完美结局（月入≥30k且满意度≥4）、良好结局（月入≥15k且满意度≥3）、苦涩结局（满意度≤2）、普通结局（其他情况）。多周目传承机制允许玩家将当前人生的属性加成（按10-15%比例）遗传给下一代，并解锁既视感能力。

2.9 人格系统

游戏内置8种高考人格类型，根据玩家在高中阶段的选择动态计算：学术型、创意型、社交型、运动型、均衡型、完美型、反叛型、后劲型。人格计算基于Intellect/Mental/Social/Health四个维度进行综合评分。

一、系统架构概述

1.1 整体架构

┌─────────────────────────────────────────────┐ │ UI Layer (界面层) │ │ HomeUI / GenderSelectUI / HighSchoolUI │ │ CollegeUI / LifeUI / ResultUI │ ├─────────────────────────────────────────────┤ │ Manager Layer (管理器层) │ │ GameManager / GameStateManager / SaveManager│ │ AdManager / ShareManager │ ├─────────────────────────────────────────────┤ │ Engine Layer (引擎层) │ │ EventEngine / TalentEngine / ScoreEngine │ │ PersonalityEngine / DejaVuEngine │ ├─────────────────────────────────────────────┤ │ Data Layer (数据层) │ │ DataLoader / ResourceHelper │ ├─────────────────────────────────────────────┤ │ Model Layer (模型层) │ │ PlayerState / GameEvent / Talent │ └─────────────────────────────────────────────┘

1.2 核心设计模式

  • 单例模式(Singleton)：所有Manager和Engine类使用Instance单例模式，确保全局唯一
  • 观察者模式(Observer)：GameManager通过event实现状态变更通知
  • 工厂模式：DataLoader作为数据工厂统一创建各类游戏数据对象
  • 策略模式：事件系统支持多种交互策略（choice/merge/sort/slider/timing/dialog）

二、核心模块设计

2.1 游戏管理器（GameManager）

游戏总控制器，负责游戏生命周期管理。主要职责包括：管理游戏全局状态（isGameRunning, isPaused）、协调各Engine和Manager之间的通信、处理场景切换逻辑、管理玩家数据读写。核心方法包括：StartNewGame()、ContinueGame()、ChangePhase()、SaveGame()、CalculateGaokaoScore()、StartNewGamePlus()。

2.2 事件引擎（EventEngine）

驱动游戏进程的核心引擎，负责事件的获取、选择和结算。核心流程：获取事件 → 检查触发条件 → 检查既视感 → 展示事件 → 玩家交互选择 → 调用ResolveEvent计算结果 → 更新属性 → 记录事件完成 → 获取下一事件 → 循环。

2.3 天赋引擎（TalentEngine）

管理12种天赋的抽卡、选择和效果计算。核心方法包括：DrawTalents()、SelectTalent()、GetCurrentTalent()、ApplyTalentEffect()、GetTalentBonus()。

2.4 成绩计算引擎（ScoreEngine）

根据玩家属性和天赋计算高考成绩。计算公式：最终成绩 = 智力基础分 × 0.7 + 心态影响分 × 0.2 + 天赋加成 + 省份难度调整。

2.5 人格引擎（PersonalityEngine）

根据玩家属性计算高考人格类型。人格判定规则：取Intellect/Mental/Social/Health四个维度的最高值，若存在多个维度相等，取优先级最高的维度对应人格。

二、Engine - 游戏引擎

2.1 EventEngine.cs - 事件引擎

public class EventEngine : MonoBehaviour
{
    private List allEvents = new List();
    private List completedEventIds = new List();
    private PlayerState playerState;
    private System.Random rng;

    public void SetPlayerState(PlayerState state)
    {
        playerState = state;
        if (state != null)
        {
            completedEventIds = new List(state.completedEvents);
            rng = new System.Random();
        }
    }

    public GameEvent GetNextEvent()
    {
        if (allEvents.Count == 0) return null;

        var candidate = allEvents
            .Where(e => !completedEventIds.Contains(e.id))
            .Where(e => MeetsCondition(e))
            .ToList();

        if (candidate.Count == 0) return null;

        return candidate[rng.Next(candidate.Count)];
    }

    public EventOutcome ResolveEvent(string eventId, string choiceId)
    {
        var evt = allEvents.FirstOrDefault(e => e.id == eventId);
        if (evt == null) return null;

        completedEventIds.Add(eventId);
        playerState.completedEvents.Add(eventId);
        playerState.RecordChoice(eventId, choiceId);

        var outcomes = evt.GetOutcomeDict();
        if (outcomes.TryGetValue(choiceId, out var outcome))
        {
            ApplyOutcome(outcome);
            return outcome;
        }
        return null;
    }
}

2.2 TalentEngine.cs - 天赋引擎

public class TalentEngine : MonoBehaviour
{
    public static TalentEngine Instance { get; private set; }
    private List allTalents = new List();
    private Talent currentTalent;

    void Awake()
    {
        if (Instance == null) { Instance = this; DontDestroyOnLoad(gameObject); }
        else { Destroy(gameObject); }
    }

    public List DrawTalents(int count = 3)
    {
        if (allTalents.Count == 0) return new List();
        var candidates = allTalents
            .OrderBy(t => -t.probability)
            .Take(count * 3)
            .ToList();
        return candidates.OrderBy(_ => new System.Random().Next()).Take(count).ToList();
    }

    public void SelectTalent(string talentId)
    {
        currentTalent = allTalents.FirstOrDefault(t => t.id == talentId);
    }

    public Talent GetCurrentTalent() => currentTalent;

    public float GetTalentBonus(string statName)
    {
        if (currentTalent?.effects == null) return 1f;
        foreach (var effect in currentTalent.effects)
        {
            if (effect.type == "stat_multiply" && effect.statKey == statName)
                return 1f + effect.value / 100f;
        }
        return 1f;
    }
}

三、Managers - 管理器

3.1 SaveManager.cs - 存档管理器

public class SaveManager : MonoBehaviour
{
    public static SaveManager Instance { get; private set; }
    private const string SAVE_KEY = "gaokao_life_save";

    void Awake()
    {
        if (Instance == null) { Instance = this; DontDestroyOnLoad(gameObject); }
        else { Destroy(gameObject); }
    }

    private GameSaveData currentData;

    public void Save(GameSaveData data)
    {
        data.updatedAt = System.DateTimeOffset.UtcNow.ToUnixTimeSeconds();
        currentData = data;
        string json = JsonUtility.ToJson(data);
        PlayerPrefs.SetString(SAVE_KEY, json);
        PlayerPrefs.Save();
    }

    public GameSaveData Load()
    {
        string json = PlayerPrefs.GetString(SAVE_KEY, "");
        if (string.IsNullOrEmpty(json)) return null;
        currentData = JsonUtility.FromJson(json);
        return currentData;
    }

    public bool HasSaveData() { return PlayerPrefs.HasKey(SAVE_KEY); }

    public PlayerState GetCurrentPlayer() => currentData?.playerState;

    public void SetCurrentPlayer(PlayerState state)
    {
        if (currentData == null) currentData = new GameSaveData();
        currentData.playerState = state;
    }
}

四、Data - 数据层

4.1 DataLoader.cs - 数据加载器

public class DataLoader : MonoBehaviour
{
    public static DataLoader Instance { get; private set; }
    private List colleges = new List();
    private List majors = new List();
    private List provinces = new List();

    void Awake()
    {
        if (Instance == null) { Instance = this; DontDestroyOnLoad(gameObject); }
        else { Destroy(gameObject); }
    }

    public void LoadAllData(System.Action callback)
    {
        StartCoroutine(LoadAllDataRoutine(callback));
    }

    private IEnumerator LoadAllDataRoutine(System.Action callback)
    {
        yield return StartCoroutine(LoadJson("colleges", list => colleges = list));
        yield return StartCoroutine(LoadJson("majors", list => majors = list));
        yield return StartCoroutine(LoadJson("provinces", list => provinces = list));
        TalentEngine.Instance?.LoadTalents(talents);
        PersonalityEngine.Instance?.LoadPersonalities(personalities);
        callback?.Invoke();
    }

    public College GetCollegeById(string id) => colleges.FirstOrDefault(c => c.id == id);
    public Major GetMajorById(string id) => majors.FirstOrDefault(m => m.id == id);
}

4.2 ResourceHelper.cs - 资源加载助手

public static class ResourceHelper
{
    private static readonly Color DefaultPlaceholderColor = new Color(0.8f, 0.8f, 0.8f);

    public static Sprite LoadSprite(string path, int fallbackW = 128, 
                                   int fallbackH = 128, Color? fallbackColor = null)
    {
        Sprite sprite = Resources.Load(path);
        if (sprite != null) return sprite;

        var tex = Placeholder.Generate(fallbackW, fallbackH, 
            fallbackColor ?? DefaultPlaceholderColor, path);
        sprite = Sprite.Create(tex, new Rect(0, 0, tex.width, tex.height), 
            new Vector2(0.5f, 0.5f));
        return sprite;
    }

    public static Sprite LoadEventIllustration(string illustrationId, Gender gender, string stage)
    {
        string prefix = gender == Gender.Male ? "男illu" : "女illu";
        string stageFolder = stage switch
        {
            "highschool" => "事件高中",
            "college" => "事件大学",
            _ => "事件高中"
        };
        string path = $"Tex/{stageFolder}/{prefix}_{illustrationId}";
        return LoadSprite(path, 680, 380, new Color(0.9f, 0.9f, 0.95f));
    }
}

public class HomeUI : MonoBehaviour

    }
}

六、GameManager - 总控制器

public class GameManager : MonoBehaviour
{
    public static GameManager Instance { get; private set; }

    private PlayerState currentPlayer;
    private GamePhase currentPhase;
    private bool isGameRunning;

    void Awake()
    {
        if (Instance == null) { Instance = this; DontDestroyOnLoad(gameObject); }
        else { Destroy(gameObject); }
    }

    public void StartNewGame()
    {
        Gender savedGender = Gender.Male;
        if (SaveManager.Instance != null)
        {
            var existing = SaveManager.Instance.GetCurrentPlayer();
            if (existing != null) savedGender = existing.gender;
        }

        SaveManager.Instance?.CreateNewGame();
        currentPlayer = SaveManager.Instance?.GetCurrentPlayer() ?? new PlayerState();
        currentPlayer.gender = savedGender;
        isGameRunning = true;

        LoadGameData();
        LoadScene("Home");
    }

    public void ChangePhase(GamePhase newPhase)
    {
        currentPhase = newPhase;
        currentPlayer.currentPhase = newPhase;
        LoadScene(newPhase.ToString());
    }

    public int CalculateGaokaoScore()
    {
        if (ScoreEngine.Instance != null)
        {
            string talentId = TalentEngine.Instance?.GetCurrentTalent()?.id ?? "";
            return ScoreEngine.Instance.CalculateScore(currentPlayer, talentId);
        }
        return 400;
    }
}
